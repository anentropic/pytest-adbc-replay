## Blocking Issues

### Issue 1 — Version Mismatch (CRITICAL)
**Affects:** PUB-03

`pyproject.toml` `version = "0.1.0"` while `CHANGELOG.md` documents `[1.0.0]`. Pushing a `v1.0.0` tag will publish version `0.1.0` to PyPI.

**Fix:** Change `version = "0.1.0"` → `version = "1.0.0"` in `pyproject.toml` before tagging.

---

### Issue 2 — TODO GitHub URLs (CRITICAL)
**Affects:** PUB-01, PUB-03, DOC-01

`pyproject.toml [project.urls]` contains:
```
Source = "https://github.com/TODO/pytest-adbc-replay"
Issues = "https://github.com/TODO/pytest-adbc-replay/issues"
Changelog = "https://github.com/TODO/pytest-adbc-replay/blob/main/CHANGELOG.md"
Documentation = "https://TODO.github.io/pytest-adbc-replay"
```
Real username per `mkdocs.yml`: `anentropic`

`README.md` also contains one TODO doc URL.

**Fix:** Replace all `TODO` with `anentropic`.

---

### Issue 3 — actions/checkout@v6 (SIGNIFICANT)
**Affects:** PUB-02

`pr.yml` line 25 uses `actions/checkout@v6` (non-existent version — highest stable is `v4`). Coverage comment job fails on every PR. Test matrix job is unaffected (it calls `_test.yml` which correctly uses `v4`).

**Fix:** Change `actions/checkout@v6` → `actions/checkout@v4` in `pr.yml`.
